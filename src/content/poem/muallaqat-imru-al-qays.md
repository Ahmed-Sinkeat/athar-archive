---
١ - قِفا نَبكِ مِن ذِكرى حَبيبٍ وَمَنزِلِ … بِسِقطِ اللِوى بَينَ الدَخولِ فَحَومَلِ

٢ - فَتوضِحَ فَالمِقراةِ لَم يَعفُ رَسمُه … لِما نَسَجَتها مِن جَنوبٍ وَشَمأَلِ

٣ - تَرى بَعَرَ الآرامِ في عَرَصاتِه … وَقيعانِها كَأَنَّهُ حَبُّ فُلفُلِ

٤ - كَأَنّي غَداةَ البَينِ يَومَ تَحَمَّلو … لَدى سَمُراتِ الحَيِّ ناقِفُ حَنظَلِ

٥ - وُقوفًا بِها صَحبي عَلَيَّ مَطِيِّهُم … يَقولونَ لا تَهلِك أَسىً وَتَجَمَّلِ

٦ - وَإِنَّ شِفائي عَبرَةٌ مَهَراقَةٌ … فَهَل عِندَ رَسمٍ دارِسٍ مِن مُعَوَّلِ

٧ - كَدَأبِكَ مِن أُمِّ الحُوَيرِثِ قَبلَه … وَجارَتِها أُمِّ الرَبابِ بِمَأسَلِ

٨ - فَفاضَت دُموعُ العَينِ مِنّي صَبابَةً … عَلى النَحرِ حَتّى بَلَّ دَمعِيَ مِحمَلي

٩ - أَلا رُبَّ يَومٍ لَكَ مِنهُنَّ صالِحٌ … وَلا سِيَّما يَومٍ بِدارَةِ جُلجُلِ

١٠ - وَيَومَ عَقَرتُ لِلعَذارى مَطِيَّتي … فَيا عَجَبًا مِن كورِها المُتَحَمَّلِ

١١ - فَظَلَّ العَذارى يَرتَمينَ بِلَحمِه … وَشَحمٍ كَهُدّابِ الدِمَقسِ المُفَتَّلِ

١٢ - وَيَومَ دَخَلتُ الخِدرَ خِدرَ عُنَيزَةٍ … فَقالَت لَكَ الوَيلاتُ إِنَّكَ مُرجِلي

١٣ - تَقولُ وَقَد مالَ الغَبيطُ بِنا مَع … عَقَرتَ بَعيري يا اِمرَأَ القَيسِ فَاِنزُلِ

١٤ - فَقُلتُ لَها سيري وَأَرخي زِمامَهُ … وَلا تُبعِديني مِن جَناكِ المُعَلَّلِ

١٥ - فَمِثلُكِ حُبلى قَد طَرَقتُ وَمُرضِعٍ … فَأَلهَيتُها عَن ذي تَمائِمَ مُحوِلِ

١٦ - إِذا ما بَكى مِن خَلفِها اِنصَرَفَت لَهُ … بِشِقٍّ وَتَحتي شِقُّها لَم يُحَوَّلِ

١٧ - وَيَومًا عَلى ظَهرِ الكَثيبِ تَعَذَّرَت … عَلَيَّ وَآلَت حَلفَةً لَم تُخَلَّلِ

١٨ - أَفاطِمَ مَهلًا بَعضَ هَذا التَدَلُّلِ … وَإِن كُنتِ قَد أَزمَعتِ صَرمي فَأَجمِلي

١٩ - وَإِن تَكُ قَد ساءَتكِ مِنّي خِلقَةٌ … فَسُلّي ثِيابي مِن ثِيابِكِ تَنسُلِ

٢٠ - أَغَرَّكِ مِنّي أَنَّ حُبَّكِ قاتِلي … وَأَنَّكِ مَهما تَأمُري القَلبَ يَفعَلِ

٢١ - وَما ذَرَفَت عَيناكِ إِلّا لِتَضرِبي … بِسَهمَيكِ في أَعشارِ قَلبٍ مُقَتَّلِ

٢٢ - وَبَيضَةِ خِدرٍ لا يُرامُ خِباؤُه … تَمَتَّعتُ مِن لَهوٍ بِها غَيرَ مُعجَلِ

٢٣ - تَجاوَزتُ أَحراسًا إِلَيها وَمَعشَر … عَلَيَّ حِراسًا لَو يُسِرّونَ مَقتَلي

٢٤ - إِذا ما الثُرَيّا في السَماءِ تَعَرَّضَت … تَعَرُّضَ أَثناءِ الوِشاحِ المُفَصَّلِ

٢٥ - فَجِئتُ وَقَد نَضَّت لِنَومٍ ثِيابَه … لَدى السِترِ إِلّا لِبسَةَ المُتَفَضِّلِ

٢٦ - فَقالَت يَمينَ اللَهِ ما لَكَ حيلَةٌ … وَما إِن أَرى عَنكَ الغِوايَةَ تَنجَلي

٢٧ - خَرَجتُ بِها أَمشي تَجُرُّ وَراءَن … عَلى أَثَرَينا ذَيلَ مِرطٍ مُرَحَّلِ

٢٨ - فَلَمّا أَجَزنا ساحَةَ الحَيِّ وَاِنتَحى … بِنا بَطنُ خَبثٍ ذي حِقافٍ عَقَنقَلِ

٢٩ - هَصَرتُ بِفَودي رَأسِها فَتَمايَلَت … عَلَيَّ هَضيمَ الكَشحِ رَيّا المُخَلخَلِ

٣٠ - إِذا اِلتَفَتَت نَحوي تَضَوَّعَ ريحُه … نَسيمَ الصَبا جاءَت بِرَيّا القَرَنفُلِ

٣١ - مُهَفهَفَةٌ بَيضاءُ غَيرُ مُفاضَةٍ … تَرائِبُها مَصقولَةٌ كَالسَجَنجَلِ

٣٢ - كَبِكرِ المُقاناةِ البَياضِ بِصُفرَةٍ … غَذاها نَميرُ الماءِ غَيرُ المُحَلَّلِ

٣٣ - تَصُدُّ وَتُبدي عَن أَسيلٍ وَتَتَّقي … بِناظِرَةٍ مِن وَحشِ وَجرَةَ مُطفِلِ

٣٤ - وَجيدٍ كَجيدِ الرِئمِ لَيسَ بِفاحِشٍ … إِذا هِيَ نَصَّتهُ وَلا بِمُعَطَّلِ

٣٥ - وَفَرعٍ يَزينُ المَتنَ أَسوَدَ فاحِمٍ … أَثيثٍ كَقِنوِ النَخلَةِ المُتَعَثكِلِ

٣٦ - غَدائِرُها مُستَشزِراتٌ إِلى العُل … تَضِلُّ العِقاصَ في مُثَنّىً وَمُرسَلِ

٣٧ - وَكَشحٍ لَطيفٍ كَالجَديلِ مُخَصَّرٍ … وَساقٍ كَأُنبوبِ السَقِيِّ المُذَلَّلِ

٣٨ - وَتَعطو بِرَخصٍ غَيرِ شَثنٍ كَأَنَّهُ … أَساريعُ ظَبيٍ أَو مَساويكُ إِسحِلِ

٣٩ - تُضيءُ الظَلامَ بِالعِشاءِ كَأَنَّه … مَنارَةُ مَمسى راهِبٍ مُتَبَتِّلِ

٤٠ - وَتُضحي فَتيتُ المِسكِ فَوقَ فِراشِه … نَؤومُ الضُحى لَم تَنتَطِق عَن تَفَضُّلِ

٤١ - إِلى مِثلِها يَرنو الحَليمُ صَبابَةً … إِذا ما اِسبَكَرَّت بَينَ دِرعٍ وَمِجوَلِ

٤٢ - تَسَلَّت عِماياتُ الرِجالِ عَنِ الصِب … وَلَيسَ فُؤادي عَن هَواكِ بِمُنسَلِ

٤٣ - أَلا رُبَّ خَصمٍ فيكِ أَلوى رَدَدتَهُ … نَصيحٍ عَلى تَعذالِهِ غَيرَ مُؤتَلِ

٤٤ - وَلَيلٍ كَمَوجِ البَحرِ أَرخى سُدولَهُ … عَلَيَّ بِأَنواعِ الهُمومِ لِيَبتَلي

٤٥ - فَقُلتُ لَهُ لَمّا تَمَطّى بِصُلبِهِ … وَأَردَفَ أَعجازًا وَناءَ بِكَلكَلِ

٤٦ - أَلا أَيُّها اللَيلُ الطَويلُ أَلا اِنجَلي … بِصُبحٍ وَما الإِصباحُ مِنكَ بِأَمثَلِ

٤٧ - فَيا لَكَ مِن لَيلٍ كَأَنَّ نُجومَهُ … بِكُلِّ مُغارِ الفَتلِ شُدَّت بِيَذبُلِ

٤٨ - كَأَنَّ الثُرَيّا عُلِّقَت في مَصامِه … بِأَمراسِ كِتّانٍ إِلى صُمِّ جَندَلِ

٤٩ - وَقَد أَغتَدي وَالطَيرُ في وُكُناتِه … بِمُنجَرِدٍ قَيدِ الأَوابِدِ هَيكَلِ

٥٠ - مِكَرٍّ مِفَرٍّ مُقبِلٍ مُدبِرٍ مَع … كَجُلمودِ صَخرٍ حَطَّهُ السَيلُ مِن عَلِ

٥١ - كُمَيتٍ يَزِلُّ اللِبدُ عَن حالِ مَتنِهِ … كَما زَلَّتِ الصَفواءُ بِالمُتَنَزَّلِ

٥٢ - مِسَحٍّ إِذا ما السابِحاتُ عَلى الوَنى … أَثَرنَ غُبارًا بِالكَديدِ المُرَكَّلِ

٥٣ - عَلى العَقبِ جَيّاشٍ كَأَنَّ اِهتِزامَهُ … إِذا جاشَ فيهِ حَميُهُ غَليُ مِرجَلِ

٥٤ - يَطيرُ الغُلامَ الخِفُّ عَن صَهَواتِهِ … وَيَلوي بِأَثوابِ العَنيفِ المُثَقَّلِ

٥٥ - دَريرٍ كَخُذروفِ الوَليدِ أَمَرَّهُ … تَقَلُّبُ كَفَّيهِ بِخَيطٍ مُوَصَّلِ

٥٦ - لَهُ أَيطَلا ظَبيٍ وَساقا نَعامَةٍ … وَإِرخاءُ سِرحانٍ وَتَقريبُ تَتفُلِ

٥٧ - كَأَنَّ عَلى الكَتفَينِ مِنهُ إِذا اِنتَحى … مَداكُ عَروسٍ أَو صَلايَةُ حَنظَلِ

٥٨ - وَباتَ عَلَيهِ سَرجُهُ وَلِجامُهُ … وَباتَ بِعَيني قائِمًا غَيرَ مُرسَلِ

٥٩ - فَعَنَّ لَنا سِربٌ كَأَنَّ نِعاجَهُ … عَذارى دَوارٍ في مُلاءٍ مُذَيَّلِ

٦٠ - فَأَدبَرنَ كَالجِزعِ المُفَصَّلِ بَينَهُ … بِجيدِ مُعَمٍّ في العَشيرَةِ مُخوَلِ

٦١ - فَأَلحَقَنا بِالهادِياتِ وَدونَهُ … جَواحِرُها في صَرَّةٍ لَم تُزَيَّلِ

٦٢ - فَعادى عِداءً بَينَ ثَورٍ وَنَعجَةٍ … دِراكًا وَلَم يَنضَح بِماءٍ فَيُغسَلِ

٦٣ - وَظَلَّ طُهاةُ اللَحمِ ما بَينَ مُنضِجٍ … صَفيفَ شِواءٍ أَو قَديرٍ مُعَجَّلِ

٦٤ - وَرُحنا وَراحَ الطَرفُ يُنفِضُ رَأسَهُ … مَتى ما تَرَقَّ العَينُ فيهِ تَسَفَّلِ

٦٥ - كَأَنَّ دِماءَ الهادِياتِ بِنَحرِهِ … عُصارَةُ حِنّاءٍ بِشَيبٍ مُرَجَّلِ

٦٦ - وَأَنتَ إِذا اِستَدبَرتَهُ سَدَّ فَرجَهُ … بِضافٍ فُوَيقَ الأَرضِ لَيسَ بِأَعزَلِ

٦٧ - أَحارِ تَرى بَرقًا أُريكَ وَميضَهُ … كَلَمعِ اليَدَينِ في حَبِيٍّ مُكَلَّلِ

٦٨ - يُضيءُ سَناهُ أَو مَصابيحَ راهِبٍ … أَهانَ السَليطَ في الذَبالِ المُفَتَّلِ

٦٩ - قَعَدتُ لَهُ وَصُحبَتي بَينَ حامِرٍ … وَبَينَ إِكامِ بُعدَ ما مُتَأَمَّلِ

٧٠ - وَأَضحى يَسُحُّ الماءُ عَنكُلِّ فَيقَةٍ … يَكُبُّ عَلى الأَذقانِ دَوحَ الكَنَهبَلِ

٧١ - وَتَيماءَ لَم يَترُك بِها جِذعَ نَخلَةٍ … وَلا أُطُمًا إِلّا مَشيدًا بِجَندَلِ

٧٢ - كَأَنَّ ذُرى رَأسِ المُجَيمِرِ غُدوَةً … مِنَ السَيلِ وَالغُثّاءِ فَلكَةُ مِغزَلِ

٧٣ - كَأَنَّ أَبانًا في أَفانينِ وَدقِهِ … كَبيرُ أُناسٍ في بِجادٍ مُزَمَّلِ

٧٤ - وَأَلقى بِصَحراءِ الغَبيطِ بَعاعَهُ … نُزولَ اليَماني ذي العِيابِ المُخَوَّلِ

٧٥ - كَأَنَّ سِباعًا فيهِ غَرقى غُدَيَّةً … بِأَرجائِهِ القُصوى أَنابيشُ عَنصُلِ

٧٦ - عَلى قَطَنٍ بِالشَيمِ أَيمَنُ صَوبِهِ … وَأَيسَرُهُ عَلى السِتارِ فَيَذبُلِ

٧٧ - وَأَلقى بِبَيسانَ مَعَ اللَيلِ بَركَهُ … فَأَنزَلَ مِنهُ العَصمَ مِن كُلِّ مَنزِلِ
