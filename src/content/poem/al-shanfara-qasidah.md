١ - أَقيموا بَني أُمّي صُدورَ مَطِيَّكُم … فَإِنّي إِلى قَومٍ سِواكُم لأمْيَلُ

٢ - فَقَد حُمَّت الحاجاتُ واللَيلُ مُقمِرٌ … وَشُدَّت لِطِيّاتٍ مَطايا وَأَرُحلُ

٣ - وَفي الأَرضِ مَنأى لِلكَريمِ عَنِ الأَذى … وَفيها لِمَن خافَ القِلى مُتَعَزَّلُ

٤ - لَعَمرُكَ ما في الأَرضِ ضيقٌ عَلى اِمرئٍ … سَرى راغِباً أَو راهِباً وَهوَ يَعقِلُ

٥ - وَلي دونَكُم أَهلَونَ سيدٌ عَمَلَّسٌ … وَأَرقَطُ زُهلولٌ وَعَرفاءُ جَيأَلُ

٦ - هُمُ الرَهطُ لا مُستَودَعُ السِرَّ ذائِعٌ … لَدَيهِم وَلا الجاني بِما جَرَّ يُخذَلُ

٧ - وَكُلٌّ أَبِيٌّ باسِلٌ غَيرَ أَننَّي … إِذا عَرَضَت أُولى الطَرائِدِ أَبسَلُ

٨ - وَإِن مُدَّتِ الأَيدي إِلى الزادِ لَم أَكنُ … بِأَعجَلِهِم إِذ أَجشَعُ القَومِ أَعجَلُ

٩ - وَما ذاكَ إِلاّ بَسطَةٌ عَن تَفَضُّلٍ … عَلَيهِم وَكانَ الأَفضَلَ المُتَفَضَّلُ

١٠ - وَلي صاحِبٌ من دونِهم لا يَخونني … إِذا التبسَت كفِّي بِهِ يَتَأكّلُ

١١ - وَإِنّي كَفانِي فَقدُ مَن لَيسَ جازِياً … بِحُسنى وَلا في قُربٍه مُتَعَلَّلُ

١٢ - ثَلاثَةُ أَصحابٍ فُؤادٌ مُشَيَّعٌ … وَأَبيَضُ إِصليتٌ وَصَفراءُ عَيطَلُ

١٣ - هَتوفٌ مِنَ المُلسِ المُتونِ يَزينُها … رَصائِعُ قَد نيطَت إلَيها وَمِحمَلُ

١٤ - إِذا زَلَّ عَنها السَهمُ حَنَّت كَأَنَّها … مُرَزَّأَةٌ عَجلى تُرِنُّ وَتُعوِلُ

١٥ - وَأَغدو خميصَ البَطنِ لا يَستَفِزُّني … إِلى الزاد حِرصٌ أَو فُؤادٌ مُوَكّلُ

١٦ - وَلَستُ بِمِهيافٍ يُعَشّي سَوامَهُ … مُجَدَّعَةً سُقبانَها وَهيَ بُهَّلُ

١٧ - وَلا جَبأَ أَكهى مُرِبٍّ بِعِرسِهِ … يُطالِعُها في شَأنِهِ كَيفَ يَفعَلُ

١٨ - وَلا خَرِقٍ هَيقٍ كَأَنَّ فُؤادَهُ … يَظَلُّ بِهِ المُكَّاءُ يَعلو وَيَسفِلُ

١٩ - وَلا خالِفٍ دارِيَّةٍ مُتَغَزَّلٍ … يَروحُ وَيَغدو داهِناً يَتَكَحَّلُ

٢٠ - وَلَستُ بِعَلٍّ شَرُّهُ دونَ خَيرِهِ … أَلَفَّ إِذا ما رُعتَهُ اِهتاجَ أَعزَلُ

٢١ - وَلَستُ بِمِحيارِ الظَلامِ إِذا اِنتَحَت … هُدى الهَوجِل العِسّيفِ يَهماءُ هَوجَلُ

٢٢ - إِذا الأَمَعزُ الصُوّانُ لاقى مَناسِمي … تَطايَرَ مِنهُ قادِحٌ وَمُفَلَّلُ

٢٣ - أَديمُ مِطالَ الجوعِ حَتّى أُميتَهُ … وَأَضرِبُ عَنهُ الذِكرَ صَفحاً فَأَذهَلُ

٢٤ - وَأَستَفُّ تُربَ الأَرضِ كَيلا يَرى لَهُ … عَلَيَّ مِنَ الطَولِ اِمُرؤ مُتَطَوَّلُ

٢٥ - وَلَولا اجتِنابُ الذَأم لَم يُلفَ مَشرَبٌ … يُعاشُ بِهِ إِلّا لَدَيَّ وَمَأكَلُ

٢٦ - وَلَكِنَّ نَفساً مُرَّةً لا تُقيمُ بي … عَلى الذَأمِ إِلّا رَيثما أَتَحَوَّلُ

٢٧ - وَاَطوي عَلى الخُمصِ الحَوايا كَما اِنطَوَت … خُيوطَةُ مارِيٍّ تُغارُ وَتُفتَلُ

٢٨ - وَأَغدو عَلى القوتِ الزَهيدِ كَما غَدَا … أَزَلُّ تَهاداهُ التَنائِفُ أَطحَلُ

٢٩ - غَدا طاوِياً يُعارِضُ الريحَ هافِياً … يَخوتُ بِأَذنابِ الشِعابِ وَيَعسَلُ

٣٠ - فَلَمّا لَواهُ القوتُ مِن حَيثُ أَمَّهُ … دَعا فَأَجابَتهُ نَظائِرُ نُحَّلُ

٣١ - مُهَلهَةٌ شِيبُ الوُجوهِ كَأَنَّها … قِداحٌ بِكَفَّي ياسِرٍ تَتَقَلقَلُ

٣٢ - أَو الخَشرَمُ المَبعوثُ حَثحَثَ دَبرَهُ … مَحابيضُ أَرداهُنَّ سامٍ مُعَسَّلُ

٣٣ - مُهَرَّتَهٌ فوهٌ كَأَنَّ شُدوقَها … شُقوقُ العِصِيَّ كَالِحاتٌ وَبُسلُ

٣٤ - فَضَجَّ وَضَجَّت بِالبَراح كَأَنَّها … وَإِيّاه نوحٌ فَوقَ عَلياءَ ثُكَّلُ

٣٥ - وَأَغضى وَأَغضَت وَاِتَّسى وَاِتَّسَت بِهِ … مَراميلُ عَزّاها وَعَزَّتهُ مُرمِلُ

٣٦ - شَكا وَشَكَت ثُمَّ اِرعَوى بَعدُ وَاِرعَوَت … وَلَلصَّبرُ إِن لَم يَنفَع الشَكوُ أَجمَلُ

٣٧ - وَفاءَ وَفاءَت بادِراتٌ وَكُلُّهَا … عَلى نَكَظٍ مِمّا يُكاتِمُ مُجمِلُ

٣٨ - وَتَشرِبُ أَسآرِيَ القَطا الكُدرُ بَعدَما … سَرَت قَرَباً أَحناؤُها تَتَصَلصَلُ

٣٩ - هَمَمتُ وَهَمَّت وَاِبتَدَرنا وَأَسدَلَت … وَشَمَّرَ مِنّي فارِطٌ مُتَمَهَّلُ

٤٠ - فَوَلَّيتُ عَنها وَهَي تَكبو لِعَقرِهِ … يُباشِرُهُ مِنها ذُقونٌ وَحَوصَلُ

٤١ - كَأَنَ وَغاها حَجرَتَيهِ وَحَولَهُ … أَضاميمُ مِن سِفرِ القَبائِلِ نُزَّلُ

٤٢ - تَوافَينَ مِن شَتّى إِلَيهِ فَضَمَّها … كَما ضَمَّ أَذوادَ الأَصاريمِ مَنهَلُ

٤٣ - فَعَبَّت غِشاشاً ثُمَّ مَرَّت كَأَنَّها … مَعَ الصُبحِ رَكبٌ مِن أُحاضَةَ مُجفِلُ

٤٤ - وَآلَفَ وَجهَ الأَرضِ عِندَ افتِراشِها … بِأَهدَأ تُنبيهِ سَناسِنُ قُحَّلُ

٤٥ - وَأَعدِلُ مَنحوضاً كَأَنَّ فُصوصَهُ … كِعابٌ دَحاها لاعِبٌ فَهيَ مُثَّلُ

٤٦ - فَإِن تَبتَئِس بِالشَنفَرى أُمُّ قَسطَلٍ … لَما اِغتَبَطَت بِالشَنفَرى قَبلُ أَطوَلُ

٤٧ - طَريدُ جِناياتٍ تَياسَرنَ لَحمَهُ … عَقيرَتُهُ لأَيَّها حُمَّ أَوَّلُ

٤٨ - تَنامُ إِذا ما نامَ يَقظى عُيونُها … حِثاثاً إِلى مَكروهِهِ تَتَغَلغَلُ

٤٩ - وَإِلفُ هُمومٍ ما تَزالُ تَعودُهُ … عِياداً كَحُمّى الرَبعِ أَو هِيَ أَثقَلُ

٥٠ - إِذا وَرَدت أَصدَرتُها ثُمَّ إِنَّها … تَثوبُ فَتَأتي مِن تُحَيتٍ وَمِن عَلُ

٥١ - فَإِمّا تَرَيني كَاِبنَةِ الرَملِ ضاحِياً … عَلى رِقَّةٍ أَحفى وَلا أَتَنَعَّلُ

٥٢ - فَإِنّي لَمَولى الصَبرِ أَجتابُ بَزَّهُ … عَلى مِثلِ قَلبِ السِمعِ وَالحَزمَ أَفعَلُ

٥٣ - وَأُعدِمُ أَحياناً وَأَغنى وَإِنَّما … يَنالُ الغِنى ذو البُعَدةِ المُتَبَذَّلُ

٥٤ - فَلا جَزعٌ مِن خَلَّةٍ مُتَكَثَّفٌ … وَلا مَرحٌ تَحتَ الغِنى أَتَخَيَّلُ

٥٥ - وَلا تَزدَهي الأَجهالُ حِلمي وَلا أَرى … سِؤولاً بِأعقابِ الأَقاويلِ أَنمُلُ

٥٦ - وَلَيلَةِ نَحسٍ يَصطَلِيَ القَوسَ رَبُّها … وَأَقطُعَهُ اللاتي بِها يَتَنَبَّلُ

٥٧ - دَعَستُ عَلى غَطشٍ وَبَغشٍ وَصُحبَتي … سُعارٌ وَإِرزيزٌ وَوَجرٌ وَأَفكُلُ

٥٨ - فَأَيَّمتُ نِسواناً وَأَيتَمتُ آلَدَةً … وَعُدتُ كَما أَبدَأتُ وَاللَيلُ أَليَلُ

٥٩ - وَأَصبَحَ عني بِالغُمَيصاءِ جالِساً … فَريقانِ مَسؤولٌ وَآخَرُ يَسأَلُ

٦٠ - فَقالوا لَقَد هَرَّت بِلَيلٍ كِلابُنَا … فَقُلنا أَذِئُبٌ عَسَّ أَم عَسَّ فُرغُلُ

٦١ - فَلَم تَكُ إِلّا نَبأَةً ثُمَّ هَوَّمَت … فَقُلنا قَطاةٌ ريعَ أَم رِيَعَ أَجدَلُ

٦٢ - فَإِن يَكُ مِن جِنٍّ لَأَبَرحُ طارِقاً … وَإِن يَكُ أُنساً ماكَها الأُنسُ تَفعَلُ

٦٣ - وَيَومٍ مِنَ الشِعرى يَذوبُ لَوابُهُ … أَفاعيهِ في رَمضائِهِ تَتَمَلمَلُ

٦٤ - نَصَبتُ لَهُ وَجهي وَلا كِنَّ دونَهُ … وَلا سِترَ إِلّا الأَتُحَمِيَ المُرَعبَلُ

٦٥ - وَضافٍ إِذا هَبَّت لَهُ الريحُ طَيَّرَت … لَبائِدَ عَن أَعطافِهِ ما تَرَجَّلُ

٦٦ - بَعيدٌ بِمَسَّ الدُهنِ وَالفَليُ عَهدُهُ … لَهُ عَبَسٌ عافٍ مِنَ الغِسلِ مُحوِلُ

٦٧ - وَخَرقٍ كَضَهرِ التِرسِ قَفرٍ قَطَعتُهُ … بِعامِلَتَينِ ظَهرُهُ لَيسَ يُعمَلُ

٦٨ - وَأَلحَقتُ أولاهُ بِأُخراهُ موفِياً … عَلى قُنَّةٍ أُقعي مِراراً وَأَمثِلُ

٦٩ - تَرودُ الأَراوِيَ الصُحمُ حَولِي كَأَنَّها … عَذارى عَلَيهنَّ المُلاءُ المُذَيَّلُ

٧٠ - وَيَركُدنَ بالآصالِ حَولِي كَأَنَّني … مِنَ العُصم أَدفى يَنتَحي الكَيحَ أَعقَلُ
