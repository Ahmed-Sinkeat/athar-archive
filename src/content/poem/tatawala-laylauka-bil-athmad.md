١ - تَطاوَلَ لَيلُكَ بِالأَثمَدِ … وَنامَ الخَلِيُّ وَلَم تَرقُدِ

٢ - وَباتَ وَباتَت لَهُ لَيلَةٌ … كَلَيلَةِ ذي العائِرِ الأَرمَدِ

٣ - وَذَلِكَ مِن نَبَإٍ جاءَني … وَخُبِّرتُهُ عَن أَبي الأَسوَدِ

٤ - وَلَو عَن نَثاً غَيرِهِ جاءَني … وَجُرحُ اللِسانِ كَجُرحِ اليَدِ

٥ - لَقُلتُ مِنَ القَولِ ما لا يَزا … لُ يُؤثِرُ عَنّي يَدَ المُسنِدِ

٦ - بِأَيِّ عَلاقَتِنا تَرغَبونَ … أَعَن دَمِ عَمروٍ عَلى مَرثِدِ

٧ - فَإِن تَدفُنوا الداءَ لا نُخفِهِ … وَإِن تَبعَثوا الحَربَ لا نَقعُدِ

٨ - فَإِن تَقتُلونا نُقَتِّلُكُم … وَإِن تَقصِدوا لِدَمٍ نَقصُدِ

٩ - مَتى عَهدُنا بِطِعانِ الكُما … ةِ وَالحَمدِ وَالمَجدِ وَالسُؤدُدِ

١٠ - وَبَنيِ القِبابِ وَمَلءِ الجِفا … نِ وَالنارِ وَالحَطَبِ المُفأَدِ

١١ - وَأَعدَدتُ لِلحَربِ وَثّابَةً … جَوادَ المَحَثَّةِ وَالمِروَدِ

١٢ - سَموحاً جَموحاً وَإِحضارُها … كَمَعمَعَةِ السَعَفِ الموقَدِ

١٣ - وَمَشدودَةَ السَكِّ مَوضونَةٍ … تَضاءَلُ في الطَيِّ كَالمِبرَدِ

١٤ - تَفيضُ عَلى المَرءِ أَردانُها … كَفَيضِ الأَتِيِّ عَلى الجَدجَدِ

١٥ - وَمُطَّرِداً كَرِشاءِ الجَرو … رِ مِن خُلُبِ النَخلَةِ الأَجرَدِ

١٦ - وَذا شَطَبٍ غامِضاً كَلمُهُ … إِذا صابَ بِالعَظمِ لَم يَنأَدِ
